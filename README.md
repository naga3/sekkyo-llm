# 説教先生（Sekkyo-1）

何を言っても**上から目線で説教してくる**チャットAI。
本物の言語モデルは使っていません。キーワードと文型のパターンマッチだけで動く「偽LLM」です。

## 使い方

`index.html` をブラウザで開くだけ。サーバーも `npm install` も不要です。

## 仕組み

返答は次の3パーツをプールからランダムに選んで連結するだけ。

```
[前置き] + [本題の説教] + [追い打ち]
```

- ユーザーの発言を「あいさつ／感謝／質問」などの**文型**で判定
- 「仕事・恋愛・お金・プログラミング…」などの**キーワード**に反応してテーマ別の説教を返す
- どれにも当たらなければ、発言を引用して汎用の説教でいなす

説教のネタを増やすには、`index.html` 内の `themes` / `openers` / `closers` に文章を足すだけです。
